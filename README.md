# Локальный журнал задач

Минимальное локальное веб-приложение для рабочих задач, заметок и ссылок на файлы/папки. Работает без интернета и без внешних Python-библиотек. Желательно Python 3.8 или новее.

## Запуск на Windows

1. Скопируйте папку проекта на компьютер.
2. Дважды нажмите `run.bat`.
3. Откройте адрес, который появится в консоли: обычно `http://127.0.0.1:8787`.

Если Python не привязан к команде `python`, попробуйте:

```bat
py app.py
```

## Данные

- Основная база: `data/tasks.sqlite`
- Ежедневные резервные копии: `backups/tasks-YYYY-MM-DD.sqlite`
- Ручной бэкап можно сделать кнопкой `Бэкап` в интерфейсе.
- JSON-выгрузка доступна кнопкой `JSON`.

## Как пользоваться

- Проекты переключаются верхними вкладками.
- Для каждого проекта приложение запоминает последнюю открытую задачу.
- У задачи есть статус, краткое описание, следующий шаг и заметки.
- К задаче можно добавлять ссылки с типами: код, документы, схемы, тикет и любые свои типы.
- Кнопка `Открыть` открывает файл, папку или веб-ссылку через локальный Python-сервер.

Сетевые пути можно вставлять как есть, например:

```text
////100.100.100.100/папка/папка/файл.txt
```

На Windows приложение перед открытием приведет такой путь к UNC-виду.

## Локальная сеть

По умолчанию приложение доступно только на текущем компьютере:

```text
http://127.0.0.1:8787
```

Для доступа из локальной сети можно запустить так:

```bat
python app.py --host 0.0.0.0
```

На этом этапе авторизации нет, поэтому такой режим лучше включать только в доверенной сети.
